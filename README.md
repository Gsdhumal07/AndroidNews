# Android News App

This is a simple RSS feed reader app that currently reads my blog's feed. 

![](screenshots/Android_News_Overview.gif)

## Features
- Bookmark article
- Mark article as read
- Search articles

## Requirements
- Android Studio Bumblebee or later

## Tech Stack
- MVVM Architecture
- Jetpack Compose
- Ktor Client
- Room Database
- Coil
- XmlPullParser
- Compose Navigation
- Webview
- Kotlin Coroutines & Flow
